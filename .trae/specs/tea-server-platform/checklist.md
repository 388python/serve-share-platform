# Checklist

## 认证系统
- [ ] LinuxDo Connect OAuth 是唯一登录方式，前端无密码登录入口
- [ ] OAuth 回调正确创建/更新用户记录
- [ ] 新用户首次登录自动赠送核时
- [ ] 管理员可通过 /admin-login/?username=xxx&password=xxxx 登录
- [ ] 管理员错误凭据返回 401
- [ ] 用户会话正确管理（登录/登出）

## 数据库
- [ ] 所有表按设计正确创建
- [ ] 外键关系和索引正确配置
- [ ] SSH 密钥加密存储

## 服务器贡献
- [ ] 用户可提交服务器贡献（IP、SSH密钥、规格、倍率、过期时间等）
- [ ] 核时按公式正确计算：CPU核心数×CPU倍率×全局CPU倍率 + 内存GB×内存倍率×全局内存倍率 + 宽带Mbps×宽带倍率×全局宽带倍率 + 磁盘GB×磁盘倍率×全局磁盘倍率
- [ ] 服务器状态正确管理（待审核→运行中→已过期/已下线）

## SSH 代理与端口转发
- [ ] 端口分配管理器正常工作
- [ ] 用户通过转发端口连接时不暴露原始服务器 IP
- [ ] 虚拟机过期后端口正确回收

## 代理套件
- [ ] 服务器贡献成功后自动通过 SSH 安装代理套件
- [ ] 代理套件支持 LXD/KVM 虚拟化管理

## 虚拟机管理
- [ ] 用户可创建虚拟机（选择服务器、规格、时长）
- [ ] 虚拟机时长不超过服务器过期时间
- [ ] 虚拟机过期自动回收

## 管理后台
- [ ] 站点名称可修改且全站生效
- [ ] 签到/免费套餐开关正常工作
- [ ] 注册开关正常工作（关闭后新用户无法登录）
- [ ] 邀请码注册开关正常（开启后需邀请码）
- [ ] 充值倍率/手续费设置生效
- [ ] 虚拟化方式选择（LXD/KVM）限制用户贡献选择
- [ ] 选机模式（自动选机/机器广场）切换正常
- [ ] 核时码可生成且可兑换
- [ ] 订阅码可生成且每日自动发放
- [ ] 核时套餐可创建/编辑/删除
- [ ] 新用户赠送核时数量可设置

## LDC 支付
- [ ] 易支付兼容接口（MD5签名）订单创建成功
- [ ] 支付回调验签正确并更新核时余额

- [ ] 商户分发接口正常
