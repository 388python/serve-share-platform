# Checklist: 茶的服务器公益站

## 认证系统
- [ ] OAuth 登录为唯一登录方式，无密码登录入口
- [ ] OAuth 回调正确创建/匹配用户
- [ ] 管理员 `/admin-login` 路径可用
- [ ] 注册开关关闭时新用户无法注册
- [ ] 邀请码模式开启时需有效邀请码才能完成注册

## 服务器贡献
- [ ] 贡献表单包含所有必填字段（IP、SSH端口、SSH密钥、CPU、内存、带宽、磁盘）
- [ ] 赠金开关可用
- [ ] 核时倍率（CPU/内存/带宽/磁盘）可设置
- [ ] 过期时间可选择
- [ ] 虚拟化方式选择在管理员设置 LXD/KVM 后可用
- [ ] 贡献后自动安装 agent

## 核时计算
- [ ] 公式计算正确：CPU×CPU倍率×全局CPU倍率 + 内存×内存倍率×全局内存倍率 + 带宽×带宽倍率×全局带宽倍率 + 磁盘×磁盘倍率×全局磁盘倍率
- [ ] 全局倍率修改后实时生效

## 开机器
- [ ] 机器运行时间不超过服务器过期时间
- [ ] 核时余额不足时拒绝开机
- [ ] 开机成功后自动启动虚拟化

## SSH 代理
- [ ] 通过平台端口可 SSH 连接
- [ ] 原始服务器 IP 不暴露给用户
- [ ] 端口分配正确

## LDC 支付
- [ ] 官方 Ed25519 签名模式充值可用
- [ ] 易支付 MD5 签名模式充值可用
- [ ] 支付回调正确更新余额
- [ ] 提现调用商户分发接口成功

## 管理后台
- [ ] 站点名称可修改且页面生效
- [ ] 签到/免费套餐开关生效
- [ ] 注册开关/邀请码开关生效
- [ ] 充值倍率/手续费正确应用
- [ ] 虚拟化方式选择正确传递到用户端
- [ ] 选机模式切换正确（机器广场 vs 自动选机）
- [ ] 全局倍率配置生效
- [ ] 新用户赠送核时正确发放

## 订阅码与核时码
- [ ] 可生成唯一兑换码
- [ ] 用户兑换后核时/套餐正确到账

## 核时套餐
- [ ] 时长+核时套餐可购买
- [ ] 累计时长套餐自动触发
- [ ] 新用户赠送核时正确发放

## 签到与免费套餐
- [ ] 签到每日限一次
- [ ] 免费套餐领取可用
- [ ] 开关控制正确

## 虚拟化 Agent
- [ ] Agent 可安装到贡献服务器
- [ ] 容器/VM 可创建、启动、停止、删除
- [ ] Agent 状态上报正常

## 项目部署
- [ ] Docker 构建成功
- [ ] 代码推送到远程仓库
